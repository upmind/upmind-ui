# UI Glossary

## Component Categories

| Category | Components |
|----------|------------|
| **Layout** | Card, Container, Separator |
| **Form** | Button, Input, Select, Checkbox, Radio |
| **Feedback** | Alert, Toast, Badge |
| **Overlay** | Dialog, Drawer, Popover, Tooltip |
| **Navigation** | Tabs, Breadcrumb, Menu |
| **Data** | Table, List, Carousel |

## CVA Terms

| Term | Definition |
|------|------------|
| **CVA** | Class Variance Authority - variant management |
| **Variant** | Component style variation (primary, secondary) |
| **Size** | Dimension variant (sm, md, lg) |
| **Compound Variant** | Combined variant conditions |
| **Default Variants** | Fallback variant values |

## File Naming

| Pattern | Purpose |
|---------|---------|
| `{Component}.vue` | Main component |
| `{Component}.styles.ts` | CVA definitions |
| `{Component}Trigger.vue` | Trigger sub-component |
| `{Component}Content.vue` | Content sub-component |
